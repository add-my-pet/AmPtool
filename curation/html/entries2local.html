entries2local

copies all of add_my_pet/entries to local directoy entries

Contents

  • Syntax
  • Description
  • Remarks
  • Example

Syntax

entries2local

Description

copies the 4 m-files of add_my_pet/entries to local directoy entries and stores download date in file entries/date_download.mat. Creates local directory entries, if not existing

Remarks

about 250 Mb and needs updating after each change in add_my_pet/entries on the web.

Example

entries2local
