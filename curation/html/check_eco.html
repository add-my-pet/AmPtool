check_eco

Checks eco labels

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

info = check_eco(varargin)

Description

Checks eco labels as specified in get_eco for existence in global eco_types.

Input:

  • varargin: optional character or cell string with name(s) of entry (default: select('Animalia'))

Output

  • info: boolean for all codes exist (1), or not (0)

Remarks

ecotypes for the entries are read from metaData outputted from mydata-files. If not present, the mydata files are copied from the AmP site run get_eco_types to copy AmPeco info to labels for eco-codes. If eco codes are not recognized, change the codes in get_eco, or add codes to AmPeco.html.

Example of use

check_eco(select('Actinopterygii')) or check_eco('Daphnia_magna')
