repair_ecoCode

loads a mydata-file in the Matlab editor and saves files

Contents

  • Syntax
  • Description
  • Remarks
  • Example

Syntax

repair_ecoCode(my_pet)

Description

loads mydata_my_pet to allow editing ecoCodes, saves results_my_pet.mat, my_pet_res.html, my_pet_date.zip and updates allStat and saves allStat and AmPdata.zip

Input:

  • my_pet: string with name of existing entry

Remarks

Still needs syncing with server afterwards

Example

repair_ecoCode('Ceriodaphnia_dubia')
