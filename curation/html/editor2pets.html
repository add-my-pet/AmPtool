editor2pets

get pet names from files open in Matlab editor

Syntax

pets = editor2pets (i)

Description

get pet names from files open in Matlab editor of type mydata, pars_init, predict or run

Input

  • i: optional type number 1,.,4 (default 1)

Output

  • cell string with names of pets

Example of use

nm = editor2pets;
