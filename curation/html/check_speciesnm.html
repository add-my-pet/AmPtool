check_speciesnm

Checks species name and lineage

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

check_speciesnm(my_pet)

Description

Checks species name and lineage: family, order, class, phylum. Checking gives priority to AmP classification, but mentions deviation for CoL classification.

Input:

  • my_pet: character sring with name of entry

Remarks

Checks with Catalog of Life

Example of use

check_speciesnm('Turdus_merula')
