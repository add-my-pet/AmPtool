prt_taxa

writes taxa.html with jpg's of nodes

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

prt_taxa

Description

Writes taxa.html in directory add_my_pet with taxon jpg's, where taxon. The toolbar_tax.html, written by prt_toolbar has search/selection options. A click on a thumbnail opens species_tree.html at the clicked taxon, where you can see the name of the species and its phylogenetic position. Refresh the taxa-page by clicking on the logo-icon.

Output:

  • writes files ../../add_my_pet/taxa.html
  • writes files ../../add_my_pet/sys/taxaSel.html

Remarks

The page show all taxa (genus-level and higher), but the search-dropdown only allows selection above the genus-level.

Example of use

prt_taxa, which results in add_my_pet/taxa.html
