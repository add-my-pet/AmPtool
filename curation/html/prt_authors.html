Syntax

prt_authors

Description

Writes ../authors.html with all authors and their submitted/modified entries

Remarks

allStat.mat always has fields author and date_subm for each entry, but not always field(s) author_mod_*. If author_mod_* exists, date_mod_* must exist as well with a valid date-specification. Each author_mod_* field can have one or more authors

Example of use

prt_authors
