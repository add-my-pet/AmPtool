write_popStat

writes a structure with all population stats for all entries

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

popStat = write_popStat (varargin)

Description

collects all popStat.mat files from all add_my_pet/entries_web subdirectories of the AmP site into a single structure. Apart from a direct output, file popStat.mat is written in the parent directory of the current one. Meant to be run from AmPtool/curation

Input:

  • varargin: optional cell array with entry names, or character string with entry or node names (default 'Animalia')

Ouput:

  • popStatLoc: stucture with all population statistics of all entries

Remarks

This function uses wget under Windows powershell, to copy .mat files from the web to local.

Example of use

popStat = write_popStat;
