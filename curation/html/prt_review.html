prt_review

prints a tabular in Latex with species, data, refs

Contents

  • Syntax
  • Description
  • Remarks
  • Example

Syntax

prt_review(taxa, filenm)

Description

prints tabular in Latex with species, data, refs and and bib-file in local directory. Output files have extensions .tex and .bib, respectively.

Input:

  • taxa: string or cell array with name(s) of taxa
  • filenm: optional string with name of output files without extension; default: 'table'

Output

  • No explicit output, but files filenm.tex, filenm.bib and codes.tex are written

Remarks

Species names are sorted alphabetically. In Latex document: \include{tabular} ... \include{codes} ... \bibliography{...,tabular}

Example

prt_review('Cephalopoda')
