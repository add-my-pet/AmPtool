prt_my_pet_toolbar

prints the menu bar which is used for the pages which are specific for a given species

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

prt_my_pet_toolbar (metaData, destinationFolder)

Description

Prints my_pet_toolbar.html for presentation in my_pet_res.html, my_pet_par.html, my_pet_stat.html. Reads links-info from results_my_pet.mat, and assumes that the current directory contains that file

Input:

  • my_pet: character string with name of entry
  • destinationFolder : optional string with destination folder the files starting from AmPtool/curation are printed to (default: current folder),

Remarks

  • Indent of 2 spaces used for printing to html page
  • Edit /sys/drowdown.js for adding dropdwn's

Example of use

prt_my_pet_toolbar('Daphnia_magna') if you wish to print in the current folder
