check_entries

Checks entries for member inconsistencies

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

[tree local server] = check_entries

Description

checks tree leaves against subdirectories of entries, local and on server

Output:

  • tree, local, server: cell strings with names of entries
  • prints warnings on screen

Remarks

The root of the tree is Animalia. The dates are not checked Assumes that this function is run in dir AmPtool/curation and that entries is a sister directory of AmPtool Assumes that path to entries on server is: http://www.bio.vu.nl/thb/deb/deblab/add_my_pet/entries/

Example of use

check_entries
