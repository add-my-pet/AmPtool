prt_id

prints an html-page with all identifiers for specified AmP entries

Contents

  • Syntax
  • Description
  • Remarks
  • Example

Syntax

prt_id(pets, save)

Description

prints an html-page with all identifiers for specified AmP entries as present in results_my_pet.mat files

Input:

  • pets: cell array with names of existing entries
  • save: optional boolean for saving the table in AmPTool/curation (default 0: do not save)

Remarks

This function uses results_my_pet.mat files in local directories of add_my_pet/entries; AmP supports 23 websites: 7 general, 16 taxon-specific. Repair id of an entry with repair_id, get existing id's with get_links and new id's with get_id.

Example

prt_id(select('Crustacea'))
