prt_run_my_pet

writes run_my_pet.m

Contents

  • Description
  • Remarks
  • Example of use

Syntax

prt_my_pet_run (species, destinationFolder)

Description

Prints run_my_pet.m with script to run an estimation

Input:

  • species: string with latin name of species or taxon
  • destinationFolder : optional string with destination folder the files are printed to (default: current folder)

Remarks

uses default setting for parameter estimation and input/output

Example of use

prt_run_my_pet('Daphnia') if you wish to print in the current folder else prt_run_my_pet('Daphnia', '../../entries_run/') if you wish to print in sister dir of parent with name entries_run


Published with MATLAB® R2016b
