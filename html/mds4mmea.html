mds4mmea

classical multidimensional scaling applied to solutions of the mmea method

Contents

  • Syntax
  • Description
  • Example of use

Syntax

[y, e, parNms, dist] = mds4mmea (results)

Description

classical multidimensional scaling applied to n solutions of the mmea method for DEB parameter estimates using the sb loss function as estimation criterion. It computes the first 3 eigen-vectors and values of classical dimensional scaling

Input

  • results: char string with name of .mat file with solution set

Output

  • y: (n,3) matrix with eigen vectors
  • e: k-vector with eigen values
  • parNms: cell string with parameter names
  • dist: (n,n) distance matrix

Example of use

[y, e, parNms] = mds4mmea('results_Dipodomys_deserti');
