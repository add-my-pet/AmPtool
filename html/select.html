select

gets list of species that belongs to a taxon

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

species = select (taxon)

Description

gets all species in the add_my_pet collection that belong to a particular taxon.

Input:

  • taxon: optional character string with name of taxon (default: 'Animalia')

Output:

  • cell string with all species in the add_my_pet collection that belong to that taxon

Remarks

The root is Animalia. If chosen as taxon, an ordered list of all species in the collection results. The classification follows that of Wikipedia. See also select_01

Example of use

species = select('Animalia'); % get list of all species in the collection, taxonomically ordered fish = setdiff(select('Craniata'), select('Tetrapoda')) % select all 5 fish classes, alphabetically ordered
