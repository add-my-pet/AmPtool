get_id

gets id's of all AmP-supported websites

Contents

  • Syntax
  • Description
  • Remarks
  • Example of use

Syntax

[id, id_txt, nm] = get_id(my_pet, open, tab)

Description

Gets identifiers for all websites that AmP uses for this taxon. Some websites apply to all animals, others only for particular taxa. The lineage is taken from CoL.

Input:

  • my_pet: character string with name of a taxon
  • open: optional boolean for opening sites in browser (default: 0)
  • tab: optional boolean for opening table in browser (default: 0)

Output:

  • id: vector of cells with id's
  • id_txt: vector of cells with labels for id's
  • nm: name of accepted species (_ separated)

Remarks

Outputs empty id strings if identification was not successful.

Example of use

get_id('Passer_domesticus', 1, 1)
