# AmPtool

[![CI](https://github.com/add-my-pet/AmPtool/actions/workflows/ci.yml/badge.svg)](https://github.com/add-my-pet/AmPtool/actions/workflows/ci.yml)
[![codecov](https://codecov.io/gh/add-my-pet/AmPtool/branch/master/graph/badge.svg)](https://codecov.io/gh/add-my-pet/AmPtool)

MATLAB tools for the [Add-my-Pet (AmP)](https://www.bio.vu.nl/thb/deb/deblab/add_my_pet/) collection of Dynamic Energy Budget (DEB) model parameters across animal species.

## Requirements

- MATLAB R2021a or later
- [DEBtool_M](https://github.com/add-my-pet/DEBtool_M) on the MATLAB path
- `allStat.mat` from the [AmP data collection](https://www.bio.vu.nl/thb/deb/deblab/add_my_pet/AmPdata/)
